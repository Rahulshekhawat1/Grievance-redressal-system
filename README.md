# Grievance Redressal Platform - MERN Starter

This is a starter template for a Grievance Redressal project using:
- Frontend: React (Vite) (simple setup)
- Backend: Node.js + Express
- Database: MongoDB (Mongoose)

**How to use**
1. Install dependencies:
   - Frontend: `cd frontend && npm install`
   - Backend: `cd backend && npm install`
2. Start local dev servers:
   - Frontend: `cd frontend && npm run dev`
   - Backend: `cd backend && npm run dev`
3. Set a MongoDB connection string in `backend/.env` (or use `.env.example`).

This scaffold includes:
- A React frontend that calls backend APIs.
- An Express backend with routes for grievances and a simple user model.
- Mongoose models and example CRUD endpoints.
